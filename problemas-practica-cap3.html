Ejercicios de Química

Porcentaje de Composición

Masas Atómicas a usar: H=1, C=12, N=14, O=16, Na=23, S=32, Cl=35.5, K=39, Mn=55, Fe=56

Ejercicio 1: Agua (H₂O)

Calcula el porcentaje de composición de Hidrógeno (H) y Oxígeno (O) en una molécula de agua.

Masa Molar de H₂O: 2(1) + 16 = 18 g/mol

% Hidrógeno: (2 / 18) * 100 = 11.1%

% Oxígeno: (16 / 18) * 100 = 88.9%

Ejercicio 2: Dióxido de Carbono (CO₂)

Determina el porcentaje de composición de Carbono (C) y Oxígeno (O) en el dióxido de carbono.

Masa Molar de CO₂: 12 + 2(16) = 44 g/mol

% Carbono: (12 / 44) * 100 = 27.3%

% Oxígeno: (32 / 44) * 100 = 72.7%

Ejercicio 3: Cloruro de Sodio (NaCl)

Calcula el porcentaje de Sodio (Na) y Cloro (Cl) en la sal de mesa común.

Masa Molar de NaCl: 23 + 35.5 = 58.5 g/mol

% Sodio: (23 / 58.5) * 100 = 39.3%

% Cloro: (35.5 / 58.5) * 100 = 60.7%

Ejercicio 4: Ácido Sulfúrico (H₂SO₄)

Encuentra el porcentaje en masa de cada elemento en el ácido sulfúrico.

Masa Molar de H₂SO₄: 2(1) + 32 + 4(16) = 98 g/mol

% Hidrógeno: (2 / 98) * 100 = 2.0%

% Azufre: (32 / 98) * 100 = 32.7%

% Oxígeno: (64 / 98) * 100 = 65.3%

Ejercicio 5: Glucosa (C₆H₁₂O₆)

Calcula la composición porcentual de cada elemento en la glucosa, el azúcar simple.

Masa Molar de C₆H₁₂O₆: 6(12) + 12(1) + 6(16) = 180 g/mol

% Carbono: (72 / 180) * 100 = 40.0%

% Hidrógeno: (12 / 180) * 100 = 6.7%

% Oxígeno: (96 / 180) * 100 = 53.3%

Ejercicio 6: Óxido de Hierro (III) (Fe₂O₃)

Determina el porcentaje de Hierro (Fe) y Oxígeno (O) en el óxido de hierro (III), comúnmente conocido como herrumbre.

Masa Molar de Fe₂O₃: 2(56) + 3(16) = 160 g/mol

% Hierro: (112 / 160) * 100 = 70.0%

% Oxígeno: (48 / 160) * 100 = 30.0%

Ejercicio 7: Amoníaco (NH₃)

Calcula la composición porcentual del Nitrógeno (N) y el Hidrógeno (H) en el amoníaco.

Masa Molar de NH₃: 14 + 3(1) = 17 g/mol

% Nitrógeno: (14 / 17) * 100 = 82.4%

% Hidrógeno: (3 / 17) * 100 = 17.6%

Ejercicio 8: Permanganato de Potasio (KMnO₄)

Determina el porcentaje en masa de cada elemento en el permanganato de potasio.

Masa Molar de KMnO₄: 39 + 55 + 4(16) = 158 g/mol

% Potasio: (39 / 158) * 100 = 24.7%

% Manganeso: (55 / 158) * 100 = 34.8%

% Oxígeno: (64 / 158) * 100 = 40.5%

Ejercicio 9: Etanol (C₂H₅OH)

Calcula la composición porcentual de Carbono, Hidrógeno y Oxígeno en el etanol.

Masa Molar de C₂H₆O: 2(12) + 6(1) + 16 = 46 g/mol

% Carbono: (24 / 46) * 100 = 52.2%

% Hidrógeno: (6 / 46) * 100 = 13.0%

% Oxígeno: (16 / 46) * 100 = 34.8%

Ejercicio 10: Bicarbonato de Sodio (NaHCO₃)

Encuentra el porcentaje de composición de cada elemento en el bicarbonato de sodio.

Masa Molar de NaHCO₃: 23 + 1 + 12 + 3(16) = 84 g/mol

% Sodio: (23 / 84) * 100 = 27.4%

% Hidrógeno: (1 / 84) * 100 = 1.2%

% Carbono: (12 / 84) * 100 = 14.3%

% Oxígeno: (48 / 84) * 100 = 57.1%
